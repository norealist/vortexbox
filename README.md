**VortexBox** - ваш файловый сервер, в котором контролируете файлы только вы. Вы сами настраиваете сервер, но настройка не будет сложной. 

Главная киллер-фича - гибридное шифрование файлов, которые шифруются симметричным ключом, а этот ключ шифруется публичным парным ключом (RSA)
